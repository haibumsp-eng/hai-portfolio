# Hoang Hai — Videography Portfolio

Personal portfolio website for Hoang Hai, a videographer and director based in Hanoi, Vietnam.

## Pages
- **Home / About** — introduction, skills, and services
- **My Work** — project showcase with filterable video cards
- **Contact** — contact information and social links

## Tech Stack
- Vanilla HTML, CSS, JavaScript — no frameworks
- Deployed on Vercel

## Local Development
No build step required. Open any HTML file directly in a browser, or use a local server:
```
npx serve .
```

## Project Structure
```
/
├── index.html          # Home / About page
├── about-work.html     # My Work page
├── contact.html        # Contact page
├── /assets
│   ├── /images         # Photos
│   └── /thumbnails     # Video thumbnails
└── /ref                # Design tokens (colors, fonts, spacing)
```
